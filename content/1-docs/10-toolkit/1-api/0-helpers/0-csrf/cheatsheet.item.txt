Title: csrf($check = null)

----

Excerpt: Generates and checks CSRF tokens

----

Params: - 
  name: $check
  type: string
  text: Token to check, will return boolean if set

----

Return:

type: string | boolean
text:

----

Text:

## Example

### Token generation

```php
<input type="hidden" name="csrf" value="<?php echo csrf() ?>">
```

### Checking the token

```php
$token = get('csrf');
if(csrf($token)) {
  // Success
} else {
  // Token doesn't match
}
```
